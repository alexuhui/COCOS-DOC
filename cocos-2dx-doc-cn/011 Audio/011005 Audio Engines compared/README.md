#### 你的音频需求更为高级吗？
到目前为止，我们只是谈论了SimpleAudioEngine。对于大多数游戏来说，SimpleAudioEngine提供了它们所需的所有功能。如果你的音频需求超出了SimpleAudioEngine的提供范围，Cocos2d-x还提供了第二个选择，称为AudioEngine。
